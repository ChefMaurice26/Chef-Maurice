CHEF MAURICE'S KITCHEN — VERSION 6.1 VERIFIED BUILD

IMPORTANT
This build uses a fresh storage namespace so an older saved menu cannot hide the new breakfast controls.
When the app opens, look for the orange “v6.1” badge in the top center.

HOW TO OPEN
1. Delete or move older extracted Chef Maurice folders so they are not confused with this one.
2. Extract Chef-Maurice-App-Production-v6.1.zip.
3. Open the folder named Chef-Maurice-App-Production-v6.1.
4. Double-click OPEN-APP.bat or index.html.

VERIFY THE BREAKFAST UPDATE
Customer side:
1. Select Menu.
2. Select Breakfast.
3. Add Breakfast Plate.
4. You should see:
   - Pancakes (2), Waffles (2), or Grits with Toast
   - Chocolate Chips, Pecans, Apple Topping, Strawberry Topping at $1 each
   - Bacon (2), Patty Sausage (2), or Ham (2)
   - Scrambled or Boiled eggs (2)
   - Optional Patty Hashbrown for $1
   - Apple Juice, Orange Juice, Tea, Coca Cola, Sprite, Rootbeer, Orange Fanta

Owner side:
1. Account > Open Dashboard.
2. Password: maurice
3. Select Breakfast Manager.
4. Add, edit, hide, mark sold out, photograph, or delete breakfast items.

This version also retains:
- View full order details
- Kitchen display system
- Combo upgrade for $2
- Up to 2 extra sides at $2 each
- Deluxe auto-selects lettuce, tomato, pickles, and onions
- Customer usernames and email addresses
- Rewards tracking and owner CSV export
- Food and book editing, pricing, photos, additions, and deletions

NEW IN VERSION 7 — OWNER ADD-ON MANAGER
- Separate Add-ons tab in the owner dashboard
- Manage combo sides, extra sides, drinks, sandwich toppings, and all breakfast choices
- Add new add-on items
- Change add-on prices
- Set an optional limited quantity
- Mark an item sold out
- Hide/discontinue an item
- Permanently delete an item
- Customer dropdowns and checkboxes automatically use the owner-managed lists

NEW IN VERSION 8
- Egg Add-ons dropdown under egg choices
- Multiple egg add-ons may be selected and removed
- Prices are automatically added
- Owner Add-on Manager includes Egg Add-ons
- Existing Version 7 data is preserved

NEW IN VERSION 9 — ADDITIONAL DRINKS
- Customers can select additional drinks using checkboxes
- Each selected drink adds its owner-configured price to the order
- Additional drinks appear in order details and kitchen tickets
- Owner Add-on Manager includes an Additional Drinks group
- Owner can add, edit, price, limit, mark sold out, hide, or delete additional drinks
- No ice-preference prompt was added
- All Version 8 features and saved data are preserved

NEW IN VERSION 9.3 — STANDALONE DRINK ORDERING
- The Drinks tab keeps the standard menu-card design
- Every drink is listed individually
- Each drink has its own price and Add button
- Customers can order drinks without choosing food
- Adding a drink sends it directly to the cart
- No food, combo, topping, or breakfast prompts appear for drink-only orders
- The old generic Drink card is removed

VERSION 10 — DRINKS TAB RESTORED AND STABILIZED
- Drinks is always visible in the category navigation
- The generic Drink menu item remains hidden
- Every owner-managed drink is listed individually
- Each drink has its own Add button
- Customers may order drinks without food
- Drink-only orders go directly to the cart
- Sold-out and limited availability states are displayed
- Existing menu, breakfast, rewards, books, kitchen display, owner controls, and add-on data are preserved

VERSION 11 — ACCOUNT RECOVERY AND PASSWORD RULES
- Rewards signup includes a secret question and answer
- Passwords require at least 6 characters, 1 number, and 1 special character
- Forgot Username flow uses email plus the secret question
- Forgot Password flow verifies username, email, and secret answer
- Customers can reset the password on their device
- The local prototype displays a simulated reset-email notice
- Real reset emails require the hosted production backend and an email provider
